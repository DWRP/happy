## Instalar

Após clonar o projeto, para Instalar as dependencias do `FRONTEND` da aplicação:

```bash
    cd web
    yarn install
```

## Executar

Após clonar o projeto, para rodar o `FRONTEND` da aplicação, após já ter instalado:

```bash
    cd web
    yarn start
```
